# تقييم تبنّي AI/ML من Filmlane

تاريخ التقييم: 2026-02-22  
النطاق: تبنّي مفاهيم `src/ai/*` و `src/ml/*` من مشروع Filmlane داخل هذا المستودع.

## خلاصة تنفيذية

- تم **تقييم المسارات تقنيًا** وتجهيز نقطة دمج backend قائمة (`/api/agent/review`) يمكن البناء عليها.
- لا يوجد مانع هندسي يمنع الإضافة، لكن **احتياج المنتج غير مُعرّف بعد** لمسارات:
  - توليد أفكار مشاهد.
  - auto-format عبر flows مخصصة.
  - local ML classifier قابل للتدريب.
- القرار الحالي: **تأجيل التنفيذ المنتجّي** حتى تثبيت متطلبات المنتج (KPIs + UX + تكلفة inference).

## وضع الجاهزية الحالي

- بنية backend الموحّدة للمراجعة والاستخراج موجودة في runtime واحد:
  - `server/file-import-server.mjs`
  - `POST /api/agent/review`
  - `POST /api/files/extract`
- العقود والتحقق تم تشديدها على الطلب/الاستجابة في نفس المسارات.
- يوجد telemetry/logging قابل للتوسعة في:
  - `src/utils/logger.ts`
  - `src/utils/file-import/extract/index.ts`

## قرار التبنّي المرحلي

1. المرحلة الحالية (مُنفذة): تقييم + تجهيز البنية + عدم تفعيل flows جديدة.
2. المرحلة التالية (مشروطة بالمنتج): اختيار Flow واحد فقط كبداية (Auto-format أو Scene Ideas).
3. المرحلة اللاحقة: تقييم ML local فقط لو هناك متطلب offline واضح أو تكلفة API تمنع التوسع.

## معايير البدء الفعلي (Definition of Ready)

- تعريف user stories واضحة لميزة AI.
- تحديد model/provider وسياسة fallback.
- تحديد حدود latency والتكلفة (SLO/SLA).
- موافقة المنتج على UX للتشغيل اليدوي/التلقائي.
